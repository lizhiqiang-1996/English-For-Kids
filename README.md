# 音频播放器微信小程序

基于微信小程序原生框架开发的音频播放器应用，支持音频列表展示、播放控制、搜索和收藏功能。

## 功能特性

### 核心功能
- 🎵 音频列表展示 - 显示音频封面、标题、作者和时长
- 🔍 音频搜索 - 根据关键词搜索音频内容
- ❤️ 收藏功能 - 收藏喜欢的音频，显示红心图标
- 🎮 播放控制 - 播放/暂停、上一首/下一首切换
- 📊 进度控制 - 显示播放进度条，支持拖动调节
- 🔊 音量控制 - 显示音量条，支持拖动调节音量
- 🔄 播放模式 - 支持顺序播放、单曲循环、随机播放

### 技术特性
- 微信小程序原生框架
- 使用微信原生音频API
- 本地存储管理用户数据
- 响应式界面设计
- 流畅的播放体验

## 快速开始

### 开发环境要求
- 微信开发者工具
- 微信小程序账号

### 安装步骤
1. 克隆项目到本地
2. 打开微信开发者工具
3. 导入项目，选择项目根目录
4. 配置小程序AppID（可选）
5. 点击编译运行

### 使用说明
1. **音频列表页**：浏览所有音频，可以搜索和收藏
2. **播放器页**：点击音频进入播放器，控制播放、调节进度音量
3. **播放模式**：点击模式按钮切换播放方式
4. **收藏功能**：点击心形图标收藏/取消收藏音频

## 技术实现

### 音频播放管理
使用微信小程序原生音频API `wx.createInnerAudioContext()` 实现音频播放控制，包括：
- 播放/暂停/停止
- 进度控制
- 音量调节
- 播放结束自动切换

### 数据持久化
使用微信小程序本地存储API `wx.setStorageSync()` 和 `wx.getStorageSync()` 保存：
- 用户收藏列表
- 播放历史记录
- 用户设置（音量、播放模式等）

### 状态管理
通过小程序全局App实例管理播放状态，实现页面间数据共享和状态同步。

## 注意事项

1. **音频资源**：默认使用示例音频URL，实际使用时请替换为合法音频资源
2. **网络权限**：确保小程序配置中已添加音频播放相关权限
3. **用户体验**：建议在WiFi环境下播放音频，避免用户流量消耗
